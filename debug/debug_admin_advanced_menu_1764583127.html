Status

You can see the basic informations on this page.

Mobile
ISParrow

The name of the internet service provider.

Network Typearrow

The current network type of the internet service.

Bandarrow

The band the network is working on.

Dataarrow

The total data used. If Data Limit is enabled, the total data allowance and the data left will be displayed here.

Upload Speedarrow

The current upload speed.

Download Speedarrow

The current download speed.

SIM Card Statusarrow

The current status of the SIM card.

Signal Strengtharrow

The current signal strength of the network.

RSRParrow

The current RSRP (Reference Signals Received Power) value. It is a measurement of the received power level in an LTE cell network.

RSRQarrow

The current RSRQ (Reference Signal Receiving Quality) value. It is a measurement of the received quality.

SNRarrow

The current SNR (Signal to Noise Ratio) value. It is a measure that compares the level of a desired signal to the level of background noise.

MCCarrow

Mobile Country Code. Indicates the home country of the SIM card or current network.

MNCarrow

Mobile Network Code. Indicates the home region of the SIM card or current network.

IMSIarrow

International Mobile Subscriber Identity. An IMSI is a number that identifies a cellphone plan subscriber. Usually stored on SIM card.

ICCIDarrow

Integrated Circuit Card ID , a globally unique serial number that identifies the SIM card. a 19- or 20-digit number that’s typically printed on the back of a SIM card.

MSISDNarrow

Mobile Station Integrated Services Digital Network. It is the phone number which identifies a device during calls or data sessions.

Cell IDarrow

Cell Identifier, the Cell ID of the current cell.

PCIarrow

Physical Cell Identifier, PCI of the current cell.

TACarrow

Tracking Area Code, the TAC of the current cell.

CQIarrow

Channel Quality Indication, an indicator conveying the data on communication channel quality.

PMIarrow

Precoding Matrix Indicator.

TBSarrow

Transport Block Size.

RBSarrow

The number of resource blocks.

UL MCSarrow

Uplink MCS(Modulation and Coding Scheme)，higher MCS means higher transmission efficiency, which is related to network signal quality.

DL MCSarrow

Downlink MCS(Modulation and Coding Scheme)，higher MCS means higher transmission efficiency, which is related to network signal quality.

Uplink Bandwidtharrow

The bandwidth which is used for transmission of signals from the device to the base station.

Downlink Bandwidtharrow

The bandwidth which is used for transmission of signals from the base station to the device.

Uplink Frequencyarrow

The frequency which is used for transmission of signals from the device to the base station.

Downlink Frequencyarrow

The frequency which is used for transmission of signals from the base station to the device.

PUCCH TXParrow

Transmission Power of the PUCCH (Physical Uplink Control Channel). It is a measurement of the transmission power of the device.

E-ARFCNarrow

E-UTRA ARFCN(Absolute Radio Frequency Channel Number).

SS-RSRParrow

SS reference signal received power. It is a measurement of the received power level in a NR cell network.

SS-RSRQarrow

SS reference signal received quality. It is a measurement of the received quality in a NR cell network.

SS-SINRarrow

SS Signal to Interference and Noise Ratio.

NR-ARFCNarrow

5G NR ARFCN(Absolute Radio Frequency Channel Number).
